# 🌦️ Interactive Weather Analytics Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Weather Analytics Dashboard built using **Power BI**, **Weather API**, and **Figma**. The dashboard provides real-time weather insights and multi-day forecasts for different locations through an engaging and modern user interface.

The project demonstrates the complete analytics workflow from **API data collection**, **data modeling**, **data transformation**, **UI design**, and **interactive dashboard development**.

---

## 🚀 Features

* 🌍 Multi-location weather analysis
* 🌡️ Current temperature monitoring
* 💧 Humidity tracking
* 🌬️ Wind speed analysis
* 👀 Visibility metrics
* ☀️ UV Index monitoring
* 🌧️ Precipitation insights
* 📈 Temperature trend forecasting
* 🌦️ Rain probability visualization
* 🌫️ Air Quality Breakdown (CO, NO₂, O₃, PM10, PM2.5, SO₂)
* 🌅 Sunrise & Sunset information
* 🎨 Dynamic weather icons that automatically change based on selected location and weather conditions
* 🎛️ Interactive slicers for location selection
* 🌙 Modern dark-themed UI inspired by weather applications

## 📊 Data Pipeline

### 1. Data Extraction

Weather data was collected using a Weather API for multiple locations.

### 2. Data Transformation

The raw API responses were transformed and organized into structured tables:

* Current Weather Table
* Forecast Day Table
* Forecast Hour Table
* Locations Table
* Master Weather Table

### 3. Data Modeling

Relationships were established between tables to enable dynamic filtering and cross-visual interactions.

### 4. Dashboard Development

Interactive visuals and KPIs were created using Power BI.

## 🎨 UI Design Process

The dashboard background and layout were first designed in **Figma** to create a modern weather-app-inspired interface.

Design Highlights:

* Dark navy theme
* Glassmorphism-inspired cards
* Dynamic weather icons
* Responsive visual layout
* High-contrast KPI cards

The Figma design was then imported into Power BI and used as the dashboard background.
